You will learn front-end and hybrid mobile development, with server-side support, for implementing a multi-platform solution. This Specialization is offered by coursera, which is one of the best e-learning platforms with accrediting certificates. The first two courses in this Specialization cover front-end frameworks: Bootstrap 4 and React. On the server-side, you’ll learn to implement NoSQL databases using MongoDB, work within a Node.js environment and Express framework, and communicate to the client side through a RESTful API. If you enrol in this Specialization, you are expected to have a prior working knowledge of HTML, CSS and JavaScript. In this Specialization, you will get to work on hands-on exercises, culminating in developing a full-fledged application at the end of each course.
